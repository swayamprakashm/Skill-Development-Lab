

## **Program 1 - Responsive Shopping Cart Web Application**


---
### **AIM**

To develop a **responsive shopping cart web application** using **CSS3**, **Flexbox**, and **Grid** for efficient and adaptive layout design.

---

### **DESCRIPTION**

This application allows users to:

* Register and log in
* Browse a product catalog
* Add items to a shopping cart
* View and manage their cart contents

The project leverages **CSS3 features** like **Flexbox** and **CSS Grid** to create a **responsive, mobile-friendly** layout that adjusts seamlessly across various screen sizes.

### **Installation & Setup**

#### **Prerequisites**

* [Node.js](https://nodejs.org/) and **npm** must be installed on your system.

#### **Steps to Run the Project**

1. **Clone the repository**:

   ```bash
   git clone [https://github.com/swayamprakashm/shopping-cart.git](https://github.com/swayamprakashm/Skill-Development-Lab/edit/main/Experiment-01_Shopping-cart-app_CSS/Shopping-cart-app]
   cd shopping-cart
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **Start the server**:

   ```bash
   node server.js
   ```

4. **Open the application in your browser**:

   ```
   http://localhost:3000
   ```
