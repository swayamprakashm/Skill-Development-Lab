Skill Development Lab

This repository contains solutions to 16 experiments from the Skill Development course, covering:

1.Responsive Web Design (CSS, Bootstrap)
2.JavaScript (Validation, ES6, API integration)
3.Java with MySQL (CRUD)
4.Java Servlet Controller
5.Session Management
6.Node.js (Custom server, REST API, JWT)
7.React Applications
8.Django Applications

Each experiment is inside its respective folder.
